Java Swing Contact Management Program
Ο χρήστης μπορεί να προσθέτει, να επεξεργάζει, να διαγράφει και να αναζητά επαφές.

Λειτουργίες
- Προβολή επαφών
- Προσθήκη επαφής
- Επεξεργασία επαφής
- Διαγραφή επαφής
- Αναζήτηση επαφών 

How to run
Clone the repository:
   git clone https://github.com/PantelisDev/ContactManagementApp.git
Open the project in Eclipse:
   File → Import → Existing Projects into Workspace
Run Main.java
